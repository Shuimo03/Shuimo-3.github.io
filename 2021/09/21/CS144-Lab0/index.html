CS144 Lab0

现在CS144 2021开课了，还好2020的代码还在GitHub上，课程的全部lab是是西安一个简单的TCP协议，自带测试。

关于环境配置，官方提供了vbox的镜像，也可以使用官方的配置脚本，直接跑一遍就好了，不会很麻烦。

GitHub - CS144/sponge: CS144 Lab Assignmentsgithub.com

Lab0比较简单，前面几个都是热身，就不细说了，跟着文档走就可以了，其中Email有坑，可以跳过不用太在意。

和CMU15-445一样，Lab0的写代码部分都是让熟悉C++语法，不过144的更简单一点，不会的查下就好了，课程要求我们使用modern C++。

Writing webget

使用cs144的库实现一个webget，效果和之前的Fetch a Web page类似，都是抓取一个网页内容。

这里参考了下面两个页面，注意HTTP报文格式，文档里面说的很清楚了。

An in-memory reliable byte stream

这里是实现一个基于内存的有限字节流，不用去管并发情况，因为只在一个线程中使用，这个字节流类似于一个队列，一头出一头进，但字节流的容量满了就不能再写入新的数据了，要注意到的一点是写操作分成了peek和pop两步。peek是从字节流中复制输出，而pop是从字节流中移除。
